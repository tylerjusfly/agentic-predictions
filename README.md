<h2 align="center">Automate your football passion with artificial intelligence.</h2>

## Setup

### Requirements

- nodejs 22.12.0 and above

### Installation

You can only install deps with npm package manager

```shell

npm install

```

or 

```shell

yarn install

```

To spin the project up locally:

```shell
npm run dev
```

or

```shell
yarn dev
```
